## 내가 맡은 역할
- 프론트: Daum API와 KakaoMap API를 이용한 그룹개설 페이지, Full Calendar API를 이용한 일정관리 페이지 구현, 유료모임 신청시 결제구현(PostOne API)
- 백엔드: 로그인한 아이디의 권한에 맞는 그룹개설 로직처리, 내 일정에 관한 CRUD 및 조건에 맞는 모임 상세보기, 신청취소, 내가만든 모임 탈퇴
